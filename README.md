# Pick-A-Title
A lightweight, interactive browser for Choice of Games and Hosted Games titles. An easy way too toggle between catalogs and filter by genre tags to find your next story !!

    Toggle between CoG and Hosted Games catalogs.

    Filter by genre tags.

    Super lightweight (just one HTML file!!).

Click here to use the browser live! https://patienceisshort.github.io/Pick-A-Title/
